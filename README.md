# Fast API Tutorial

This is a simple API made from the [Code Camp Tutorial](https://www.youtube.com/watch?v=tLKKmouUamsA).

## What was learnt

On the tuturialm I learnt the basics of an API. How to return values, receive Path Parameters, receive Query Parameters, receive both Path and Query Parameters on the same endpoint and receive Body Request. And also, I learnt the best practices of the GET, POST, PUT and DELETE Request.

## How to run this API

Alter cloning this repo:

```
git clone git@github.com:r-vilela/FastAPI-CodeCamp
```

And having UV installed, run the command below to install the dependecies:

```
uv run
```

And run the command below to run the API:

```
uv run uvicorn main:app --reload
```

Now, visit the the [API url](localhost:8000/docs), to test the API.
